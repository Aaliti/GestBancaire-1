### Usage
This is how this project can be run locally

1. Clone the project or download it

2. In **application.properties** file, replace ${dbpass} with your postgresql database password

3. Go to the project root directory

4. Run this command in your terminal ```java -jar target/gestBancaire-app.java ```

5. Navigate to your localhost in port 8080
